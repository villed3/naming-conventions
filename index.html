Rust
Functions, methods, local variables snake_case
Macros snake_case!
Types, traits, enum variants PascalCase
Static, constants SCREAMING_SNAKE_CASE
General constructors neworwith_more_details
Conversion constructors from_some_other_type
Type parameters Concise PascalCase, or:T
Lifetimes Short lowercase
Python
Functions, methods variables snake_case
Classes PascalCase
Constants SCREAMING_SNAKE_CASE
Module snake_case.py
Package lowercase
Git
Repositories kebab-case
Branches author_feature or author_module_feature where each section is in kebab-case
Default branch name main
Elixir
Functions, variables, module attributes snake_case
Atoms :snake_case
Function/macro with exception snake_case!
Function returning boolean snake_case?
Aliases (modules) CamelCase
Unused values Begin with _
Type checking functions Begin with is_
O(1) length measuring functions End with _size
O(n) length measuring functions End with _length
JavaScript
Functions, methods, variables, local constants camelCase
Boolean variable Begin with is or has
Classes, DOM component PascalCase
Private attributes and methods _camelCase
Global constant SCREAMING_SNAKE_CASE
Filename kebab-case.js
